MacOS外接显卡折腾记录

偶然看见外接GPU这种玩法, 我的MacBook也有雷电三接口, 最近正好觉得写VsCode有些小小卡顿, 也不想大费周折换个新电脑, 只想继续使用这个MacBook (强行找需求:0 ), 所以想搞一个外接显卡来玩玩.

我的硬件设备列表:

  • Macbook Pro 2018 13 inch
  • macOS Monterey 12.01
  • Razer Core V1 (RC21-0094)
  • AMD Raedon RX5700

其中的Razer Core 不同于现在正在售的Razer Core X, 它是相当早版本的一个外接显卡, 而外接显卡由于售价原因一直停留小众, 所以在网络上能找到关于它的内容是比较少的, 这也导致了我之后差点踩坑.

eGPU外接显卡

外接显卡最吸引我的一个点就是它能让本来性能很弱但很具有便携性的设备拥有同等台式机的性能, 在保持灵活性优势的同时弥补设备的性能短板.

雷电三外接显卡是一种外置式图形处理器（eGPU），由Intel和苹果公司联合开发。它可以通过Thunderbolt 3接口与苹果电脑或其他兼容设备连接，为其提供额外的图形处理能力。用户可以将雷电三外接显卡与笔记本电脑、超极本或台式电脑等设备配合使用，以提高计算机在图形处理方面的性能表现。

雷电三外接显卡通常配备独立显卡，例如Nvidia或AMD的高性能显卡，可以提供比集成显卡更出色的图形性能，能够满足一些对图形处理要求较高的应用需求，如视频编辑、3D建模、游戏等。

雷电三外接显卡具有插即用的优点，不需要安装驱动程序，可以实现即插即用。同时，它的便携性也使得用户可以轻松将其带到不同的地方使用。

但由于显卡坞昂贵的售价(主流显卡坞一般在500美元左右), 而且虽然显卡坞可以提供便携式设备的图形升级路径，但这并不是一项完美的解决方案。显卡坞本身通常是笨重的，而且用户需要安装和拆卸显卡，这可能不是一项方便的过程。 这些原因这种玩法始终属于小众, 但现在支持雷电3/雷电4/USB4的设备越来越多, 若显卡坞之后能够慢慢降价, 这个玩法可能会慢慢流行.

雷电3协议是由英特尔和苹果共同开发的一种高速传输协议，主要用于连接外部设备，例如外接显卡、外置硬盘、网络适配器等。它基于PCI Express 3.0和Thunderbolt技术，具有带宽高、速度快、延迟低等优点。

雷电3协议的最大带宽可达到40Gbps，远高于USB 3.1的带宽，这意味着可以通过雷电3接口实现快速数据传输和高速图形处理。此外，雷电3接口还支持Daisy Chain（串联）功能，使得多个外设可以通过单个Thunderbolt 3端口连接到电脑上，大大减少了接线混乱的问题

Thunderbolt3/4

雷电3协议是由英特尔和苹果共同开发的一种高速传输协议，主要用于连接外部设备，例如外接显卡、外置硬盘、网络适配器等。它基于PCI Express 3.0和Thunderbolt技术，具有带宽高、速度快、延迟低等优点。

雷电3协议的最大带宽可达到40Gbps，远高于USB 3.1的带宽，这意味着可以通过雷电3接口实现快速数据传输和高速图形处理。此外，雷电3接口还支持Daisy Chain（串联）功能，使得多个外设可以通过单个Thunderbolt 3端口连接到电脑上，大大减少了接线混乱的问题。

值得注意的是, 在雷电3协议的初期, 虽然很多笔记本电脑都具有雷电3接口, 通过USB-C接口实现Thunderbolt 3速度和功能的接口类型。它之所以被称为“半血雷电3”，是因为它仅支持Thunderbolt 3的一部分功能，而不是完全支持Thunderbolt 3的全部功能, 但Intel10代之后的cpu支持了雷电接口直连CPU, 所以Intel10代之后的设备不会再有半血雷电3接口. 对于MacBook来说, 从2016年的MacBookPro开始, 苹果就支持完整的雷电3接口, 所以外接显卡理论上不会在我的MacBook上表现出太大的性能损失.

现在新型的设备都开始支持Thunderbolt4协议, Thunderbolt 4的最大速度为40Gbps，与Thunderbolt 3相同, 但是它支持带宽的动态分配, Thunderbolt4(一般是Intel的CPU)和USB4(一般是AMD的CPU)都是支持向下兼容雷电3的.

显卡坞与MacOS的兼容性

现在主流的显卡坞应该是几乎所有笔记本包括Mac在内都支持的了, 但令我意想不到的是, 由于Razer Core 是在2016年TB3刚提出的时候生产的型号, 尚未成熟, 所以适配性较差, 由于它使用了Mac不支持的USB控制器, 导致MacOS无法识别外置显卡坞.

可以参考这个项目来解决兼容性问题:

后互联网时代网上冲浪指南

upon this rock, i will build my church.

在信息时代，我们面临着海量信息的冲击和挑战。虽然我们可以方便地获取到大量的信息，但是信息的真假和价值往往难以判断，这可能导致人们对世界的理解更加匮乏。

同时，由于信息过多、碎片化，人们更容易陷入信息孤岛，缺乏全局的视野和深度的思考，从而难以真正理解和把握复杂的现实问题。

此外，信息时代还存在着信息过载的问题，人们可能因为过多的信息而无法有效地处理和吸收，进而导致认知负荷过大，甚至产生焦虑和厌倦感。因此，需要通过适当的方法和策略，帮助人们理解和把握信息时代的复杂现实。

过载的信息会不仅会造成时间精力上的浪费, 更有可能对人们的情绪产生负面的影响, 无效的信息会淹没那些真正有意义的信息, 从而陷入信息孤岛.

针对以上的问题, 提出了一些方法供参考.

  • 做好信息筛选, 对信息源进行过滤
  • 主动性地,有针对性地和结构化地获得信息
  • 设定时间和用途, 减少不必要通知的干扰

信息筛选

互联网的普及和发展使得人们的信息获取更加便捷、高效，促进了社会信息化的进程，同时也带来了许多创新和发展机遇。此外，互联网上的一些平台和服务也在不断优化和改进，为用户提供更好的体验和服务。

但事实是, 在某种程度上, 互联网环境正在变差. 互联网上存在着大量的不良信息、虚假信息、网络暴力等问题，而且网络安全问题也日益突出。就我个人上网经历而言, 让我感受到互联网变差的原因有以下几点:

  • 不良信息和低俗内容
  • 无休止无意义的论战
  • 缺乏事实支撑的暴论
  • 没有底线的虚假信息

这说明对信息源进行评估筛选是迫切且必要的. 信息的筛选和过滤需要我们具备较强的信息识别和分析能力，同时还需要我们持续学习和关注，以便更好地获取准确、可靠和有用的信息。

以下是一些关于信息筛选的使用建议:

  1. 确定您需要的信息类型：在浏览信息之前，您需要明确自己需要的信息类型是什么，以便更容易找到相关的内容。
  2. 关注专业网站和论坛：对于某些特定领域的信息，关注专业网站和论坛可以帮助您找到更准确和专业的信息, 拒绝空泛的”个性化推荐”. 关于这两点我会在下一章展开.
  3. 使用可信赖的搜索引擎：使用可靠的搜索引擎可以帮助您找到更准确和可信的信息，例如谷歌、bing等。
  4. 借助社交媒体：很多社交媒体可以帮助您获取最新的信息，但同时也需要注意信息的来源和真实性, 也需要借助一些更有效率的方法.
  5. 确认信息的更新时间：信息的时效性很重要，您需要确认信息的更新时间以便获得最新的信息。
  6. 比较不同的信息源：不同的信息源可能会给出不同的信息和观点，因此在做出决策之前，您需要比较不同的信息源，并尝试找到共同点和不同点

下面是使用社交媒体获取信息的一些策略:

  • 善用搜索功能, 用的越多, 省的时间精力就越多, 收获就越多.
  • 尽量使浏览信息的过程变得更加特定化, 比如豆瓣小组>微博热搜, 贴吧某一个吧>知乎首页推荐
  • 如果有需要, 尽可能多地 加入一些相关社群, 群聊, 结合第一点: 善用搜索功能, 你有可能会收获一些意外的信息.
  • 时刻记住, 利用泛社交媒体获取信息并不一定准确和高效, 所以不要沉迷于此.

掌握主动权

在获取信息的时候, 不能盲目地接受无组织的混乱的信息, 而是应该有目的地获取信息, 并将得到的信息结构化地组织起来. 在浏览信息之前，您需要明确自己需要的信息类型是什么，以便更容易找到相关的内容。永远不要相信首页推荐, 这只会让你深陷信息茧房.

在决定采用某个信息源之前，请先确认其来源的可靠性和信誉度。您可以查看该信息源的专业性、知名度、历史记录等方面的信息。最好能从相关领域的专业网站获取信息, 这往往比漫无目的地从某一个社交媒体推荐页得到的信息更加准确.

信息获取渠道

一般来说, 来源门槛越高, 可信度越高, 但限制于受众数量, 它们发布的内容需要自己解读, 随之需要更高的信息理解和处理能力, 还可能需要专业的背景知识和背景信息. 同样地, 若门槛越低, 你接触到的信息可能是经过二手甚至几手的解读, 可能离原意相差甚远, 但经过解读, 可能会使源内容(例如新发布的某项政策)不再晦涩, 更好理解, 但请注意, 这里的理解永远是别人(如果你认可, 就是替你的)的理解, 不能混淆视角.

就个人经验, 门槛高低的信息源有下表:

门槛高 门槛低
研究论文和学术文献 信息发布官网 专业书籍和期刊 数据库和知识库 专业新闻媒体 搜索引擎和论坛 泛社交自媒体 社交媒体/评论区

注意, 这里的门槛高低并没有任何的感情色彩, 不必刻意地对某些渠道产生偏见. 只要它适合你, 符合你的需要, 那便是好的.

信息结构化和输出

无序的信息就像原料, 很难直接应用, 所以需要借助一定方法对信息进行组织和结构化. 结构化信息是指将信息按照一定的规则和标准进行组织和分类，以便更好地管理、存储和利用。

我建议输出内容, 但不建议自己生产内容, 因为你若不是专业人士, 生产内容将会是一件很困难的事情. 创造出有价值的内容需要有创意和想象力。创意不足可能是由于缺乏灵感、对主题不熟悉、缺乏经验等原因导致的. 同时, 生产高质量的内容需要花费大量的时间和精力，这可能会干扰你的日常工作和生活。生产特定领域的内容也需要有相关的专业知识和技能，这可能需要花费很长时间来学习和掌握。生产内容需要强大的自我激励和自律，这是为了确保你保持专注和按时完成任务。你可能费劲心思生产所谓的”原创内容” 也是某个人早已提出过的. 所以, 我建议你组织已有的内容并”输出”文档, 因为它能帮你更好地理解信息, 并在以后你需要用到的时候方便检索.

输出内容的要点如下:

  1. 确定文档的目的和主题：确定你要输出的文档的目的和主题，这可以帮助你更好地组织和筛选你已有的内容。
  2. 收集和整理你的内容：将你已有的内容集中到一个地方，并按主题、时间或其他相关性进行分类和整理。
  3. 筛选和编辑你的内容：对你的内容进行筛选和编辑，去除重复、不必要或过时的内容，以确保你的文档更加简洁和有用。
  4. 制定文档结构和大纲：制定一个文档结构和大纲，包括章节、标题、子标题和段落。这可以帮助你更好地组织和呈现你的内容。

总之，组织和输出文档需要一定的时间和精力，但这是非常重要的，可以帮助你更好地呈现你的思想和知识。始终记住，好的文档应该简洁、准确、易于理解和吸引人。同时, 输出文档的过程应该尽可能地变得painless, 也就是说, 输出文档的过程不应占用你太长的时间, 毕竟它只是一种辅助手段, 不应当成为过程中的主角, 不要舍本逐末, 丢失重点.

设定时间和用途

尽量将不同的设备用在适合它的地方, 比如用电脑(PC)处理效率要求较高, 专业性较强的任务, 手机用来浏览新闻和娱乐, 用家用主机玩游戏等. 设定不同设备的使用时间和用途可以帮助我们更好地管理我们的时间和注意力，提高我们的生产力和效率，并促进健康的数字生活。通过设定不同设备的使用时间和用途，可以帮助我们更好地管理我们的时间和注意力，从而提高我们的工作效率和生产力。通过限制特定设备的使用，可以让我们更好地专注于学习和创造，提高我们的思维能力和创造力。通过设定不同设备的使用时间和用途，可以帮助我们更好地平衡工作和娱乐，并减少长时间使用娱乐设备的时间，从而提高我们的生活质量。

有时候我们发现自己无法控制花费在某一个设备或应用上的时间, 甚至我们根本就没有注意到我们竟然花了这么多时间在玩手机/玩游戏上面. 毕竟,若要主动地减少过多地浪费在手机上的时间, 首先你必须意识到这个问题的存在. 想要减少花费在手机上的时间, 可以参考以下的建议

  1. 设定时间限制：在手机上设定时间限制是一个好的方法，可以帮助你控制使用手机的时间。你可以设置定时器，让手机在使用一段时间后自动锁定，或者使用专门的应用程序来限制使用时间。
  2. 建立使用规则：建立一些使用规则，例如只允许在特定时间或地点使用手机。这可以帮助你控制使用时间和场合。
  3. 关闭通知：手机上的通知可能会打断你的注意力，让你花费更多时间在手机上。可以将手机上的通知关闭或只保留重要的通知，以减少干扰。
  4. 删除无用的应用：删除无用的应用程序可以减少花费在手机上的时间。删除那些你不需要的应用程序，或者将它们移到手机主屏幕以外的文件夹里，可以帮助你减少使用它们的频率。
  5. 培养其他爱好：为了减少花费在手机上的时间，可以培养其他爱好。例如，可以尝试阅读书籍、参加运动或者学习新技能等。

要知道, 比起被动地接受社交媒体软件给你的无尽的所谓个人推荐的内容, 这个世界还有相当多有意义的事情值得你投入时间和精力.

如果说以上的方法对你来说效果都不明显, 你可以试试主动地将你的手机降级, 它能带来以下的好处:

  1. 较小的屏幕和较短的电池寿命可以减少使用手机的时间，从而减少对眼睛和身体的影响，有助于保护健康。
  2. 简单易用：旧版本的手机功能相对较为简单，使用也更加直观简单，不会因为过多的功能而分散注意力，可以更加专注于使用目的。
  3. 实现通知过滤, 减少不必要信息的干扰, 重要的信息能够及时的收到, 不重要的信息可以定时的处理, 达到通知过滤的目的.
